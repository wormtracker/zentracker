Instructions for using the compiled version on 64-bit Windows machines:

1. Download the Windows 64-bit version of the MCR for R2013b from the MathWorks Web site by navigating to

   http://www.mathworks.com/products/compiler/mcr/index.html

2. Run and install the downloaded MCR_R2013b_win64_installer.exe

3. You can now start zentracker.exe
